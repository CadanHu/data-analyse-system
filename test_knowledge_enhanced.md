# 数据分析系统 (Data-Analyse-System) 综合知识库测试文档

## 一、 行政与人事政策 (HR Policies)

### 1. 远程办公制度 (Remote Work Policy)
* **适用范围**：正式员工入职满 3 个月后可申请。
* **申请流程**：
    1. 提前 2 个工作日在钉钉提交“远程办公”申请。
    2. 需获得直属主管和部门经理审批。
* **核心要求**：远程期间需保持通讯工具（Slack/飞书）在线，响应时间不得超过 15 分钟。

### 2. 差旅与报销标准 (Travel & Expense)
* **国内出差**：
    * 一线城市（北京、上海、广州、深圳）：住宿标准最高 550 元/天。
    * 其他城市：住宿标准最高 350 元/天。
    * 餐补：100 元/天（无需发票）。
* **交通工具**：原则上优先选择高铁二等座。飞行时间超过 4 小时可申请经济舱。

## 二、 技术研发规范 (Technical Standards)

### 1. 数据库命名规范 (Database Naming Convention)
* **表名**：使用小写字母，单词间用下划线 `_` 分隔（如 `user_profile`）。
* **主键**：统一命名为 `id`，类型为 `BigInt`。
* **索引**：普通索引以 `idx_` 开头，唯一索引以 `uk_` 开头。

### 2. API 接口限流策略 (Rate Limiting)
* **标准用户**：每分钟最高 60 次请求。
* **企业级用户**：每分钟最高 1000 次请求。
* **超时时间**：后端服务响应超时统一设置为 30 秒。

## 三、 产品功能介绍：DataAnalyzer Pro

### 1. 核心特性
* **智能 SQL 生成**：基于 DeepSeek-V3 模型，将自然语言转换为标准 SQL。
* **多库支持**：兼容 MySQL, PostgreSQL, SQLite 及 MongoDB。
* **动态图表**：自动识别查询结果并推荐最合适的 ECharts 图表（折线图、柱状图、饼图）。

### 2. 版本历史
* **v1.0.0 (2025-10-01)**：初始版本发布，支持基础 SQL 查询。
* **v1.2.0 (2025-12-15)**：新增向量数据库集成，支持上传 PDF/TXT 进行 RAG 检索。
* **v1.3.0 (2026-02-10)**：引入用户认证系统及多租户隔离。

## 四、 常见问题 (FAQ)

**Q: 为什么我无法连接到生产环境数据库？**
A: 请检查：1. 是否已开启 VPN；2. 数据库白名单是否已添加您的 IP 地址；3. 账号密码是否过期。

**Q: 如何清空会话上下文？**
A: 根据最新系统升级，手动清空按钮已移除。系统会自动根据会话时长和 Token 限制进行智能滑动窗口管理。
